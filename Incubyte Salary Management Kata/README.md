# Incubyte Salary Management API

A production-style API to manage employee records, calculate salary deductions, and provide salary metrics.

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- SQLite
- Pytest

## Features Completed
 
- Create Employee
- Get Employee by ID
- List Employees

## Run locally
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
